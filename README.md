# UnityJam Project

## 開発環境
- **Unity Version**: 2022.3.24f1
- **Platform**: PC (Windows/Mac)

## セットアップ手順

### 1. Git LFS のインストール
このプロジェクトでは大容量ファイル（画像、音声、モデル等）の管理に **Git LFS** を使用しています。
リポジトリをクローンする前に、以下のコマンドでGit LFSをインストールしてください。

```bash
git lfs install
```

### 2. クローン
```bash
git clone https://github.com/aptmara/UnityJam.git
```

## チーム開発ルール

### シーン編集について
- **競合回避**: `Main` シーンなどの共通シーンを複数人で同時に編集しないでください。
- **推奨ワークフロー**:
    - 作業用の一時シーンを作成してテストする。
    - 可能な限り **Prefab (プレハブ)** 化し、シーンではなくプレハブを編集する。

### コミット・プッシュ
- こまめにプル (`git pull`) して、最新の状態を保ってください。
- 大きな変更を行う際は、事前にブランチを切って作業することを推奨します。

### コード規約
- `.editorconfig` を含めています。対応するエディタ（VS, Rider, VS Code等）を使用すれば自動的にフォーマットが適用されます。
